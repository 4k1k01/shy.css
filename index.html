📦shy.css

A minimalistic, tech-inspired CSS library built on JetBrains Mono.

Docs - Github

🚀 Why use it?

  • Monospace-first design with JetBrains Mono
  • Compatible with all modern browsers
  • Responsive grid system
  • Dark and light mode support
  • Code styling for dev-focused sites
  • Fast and simple
  • Compatible with highlight.js

🛠 Getting Started

Start using shy.css in your project by adding this one line inside your <head>:

<link href="https://cdn.jsdelivr.net/gh/4k1k01/shy.css@main/css/shy.min.css" rel="stylesheet">

Then explore our detailed Documentation for components, layout helpers, and best practices.

✨ Key Features

  • Lightweight: Just a few KB, no unnecessary bloat
  • Customizable variables for easy theming
  • Built-in accessibility best practices
  • Mobile-first responsive design out of the box
  • Optimized for performance and SEO
